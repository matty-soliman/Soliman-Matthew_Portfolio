# Matthew Soliman — Portfolio (local setup)

## Files
- `index.html` — page structure & content
- `style.css` — all styling (PCB/circuit-inspired theme)
- `script.js` — typing effect, mobile nav toggle, footer year

## Run it locally
1. Open the `portfolio` folder in VS Code.
2. Install the **Live Server** extension (Extensions panel → search "Live Server" → Install).
3. Right-click `index.html` → **Open with Live Server**.
4. It opens at `http://127.0.0.1:5500` and auto-refreshes when you save changes.

## Easy things to customize next
- Swap in real project screenshots inside `/images` and reference them in the project cards.
- Add links to live sites / GitHub repos on each project card.
- Add a downloadable resume PDF link in the Contact section.
- Replace hero role list in `script.js` (the `roles` array) if your focus shifts.

## When you're ready to deploy (free)
- **GitHub Pages**: push this folder to a GitHub repo, enable Pages in repo settings.
- **Netlify**: drag-and-drop the folder onto netlify.com/drop for an instant live URL.
